# Neural Tetris

An autoplay-driven Tetris web game where AI upgrades affect placement, rotation, and routing decisions across multiple boards.

## Run

Just open `index.html` in a browser.

## Deploy

Push to GitHub repository `main` branch and GitHub Pages will deploy the static files automatically via GitHub Actions.

Expected files:
- `index.html`
- `styles.css`
- `script.js`
- `progress.md`
- `.github/workflows/pages.yml`
